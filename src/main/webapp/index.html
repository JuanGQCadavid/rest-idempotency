USERS

Data types

application/json application/xml 
                    {
                       "id": 5000,
                       "name": "Pepe",
                       "age": 54
                    }
                    
                    <user>
                        <id>5000</id>                        
                        <name>Pepe</name>
                        <age>54</age>                        
                    </user>
Field Type Description
id int User identificator. E.g: 5007
name String User name. E.g: Manuel
age int User age. E.g: 47

GET Users

URL: /api/users

Http Method: GET

Accept: application/xml, application/json

GET User

URL: /api/users/{userId}

Http Method: GET

Accept: application/xml, application/json

ADD User

URL: /api/users

Http Method: POST

Accept: application/xml, application/json

Content-Type: application/xml, application/json

UPDATE User

URL: /api/users/{userId}

Http Method: PUT

Accept: application/xml, application/json

Content-Type: application/xml, application/json

ORDERS

Data Types

application/json application/xml 
                    {
                        "id": "999919_bd64fe278c6192c821a80ce6038effe8",
                        "price": 6.94,
                        "vouncher": "KKFEV3",
                        "status": 2
                    }
                    
                    <order>
                        <id>999919_bd84fe278c6152c821a80ee6038effe6</id>
                        <price>6.94</price>
                        <vouncher>KKFEV3</vouncher>
                        <status>2</status>
                    </order>
Field Type Description
id String Order identificator. E.g: 999919_bd84fe278c6152c821a80ee6038effe6
price double Order price. E.g: 175.97
vouncher String A vouncher code associated to the order. E.g: VC705R
status int Order status. It must be: 1 (CREATED), 2 (FINISHED) or 3 (CANCELED). E.g: 2

GET Orders

URL: /api/orders

Http Method: GET

Accept: application/xml, application/json

GET Order

URL: /api/orders/{orderId}

Http Method: GET

Accept: application/xml, application/json

CREATE "virtual" Order

URL: /api/orders

Http Method: POST

Accept: application/xml, application/json

UPDATE Order

URL: /api/orders/{orderId}

Http Method: PUT

Accept: application/xml, application/json

Content-Type: application/xml, application/json
